
### HTML 태그의 종류

1.div태그
```
<div>전체</div>
```

2.span 태그

```
<span>지정된 범위만</span>
```

3.img 태그
```
  <img src="./img/dog.png" height="100px" width="100" />
```


4.table태그
```
<table>
    <tr>
        <td>
        </td>
    </tr>
</table>
```


5.br태그
```
<br>
```


6.a태그
```
<a href="#">링크</a>
```

7. input 태그

```
<input type="text" />
```

```
<input type="checkbox" />
```

```
<input type="radio" />
```

8.i태그
```
<i>기울어진 글씨체<i>
```

9.b태그
```
<b>굵은 글씨체<b>
```

10. h1,h2,h3,h4,h5,h6태그

```
<h1> 제목 </h1> //h1
<h2> 제목 </h2> //h2
...

```

11. ul태그(unorder list)

```
<ul>
<li> 정렬되지 않은 리스트</li>
</ul>
```

12. ol태그(unorder list)

```
<ol>
<li> 정렬된 리스트</li>
</ol>
```
